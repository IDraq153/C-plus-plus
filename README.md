## 📖 Descripción

Este repositorio recopila las clases, ejercicios y prácticas desarrollados durante el aprendizaje de **C++**, comenzando desde los fundamentos del lenguaje hasta los conceptos básicos de **Programación Orientada a Objetos (POO)**.

Su objetivo es documentar el proceso de aprendizaje y servir como material de consulta para reforzar los conceptos fundamentales de programación en C++.

---

- 🚀 Introducción a C++
- 📝 Variables y tipos de datos
- 🔢 Operadores y expresiones
- 🔀 Estructuras de control
- 🔁 Bucles
- 📦 Funciones
- 📚 Arreglos y matrices
- 🔤 Cadenas de caracteres
- 👉 Punteros y referencias
- 🧩 Estructuras (`struct`)
- 🏛️ Clases y objetos
- 🔒 Encapsulamiento
- 🛠️ Constructores y destructores
- 🎯 Introducción a la Programación Orientada a Objetos
- 📋 Ejercicios prácticos
